# My-Old-Game
Hi this is my old game that I'm putting out to the public!
I will have to show you how to install my game and run it (doesn't save progress)
I will also give the links to somethings that will help you edit my code or help you make your own game!
Before anything please install the folder OldGame
After you do that unzip it
After you do that head to this website https://codewith.mu/en/download and download Mu
After you install Mu please run the installer (it may take a little while to open)
After it opens select Pygame or whatever it's called then click import
Select the Python Project.py folder and then you may look at my code or edit it!
To run the game click "play" and a window will open up showing you my game that I created a while ago!
